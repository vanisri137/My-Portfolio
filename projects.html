Battery Performance Prediction using Machine Learning


Developed supervised machine learning models to predict lithium-ion battery discharge capacity and cycle life using structured experimental datasets. Performed feature selection based on electrochemical parameters such as cycle number, current density, voltage window, and internal resistance to model degradation behavior. Implemented regression algorithms including Linear Regression and Random Forest to capture performance trends and nonlinear degradation patterns. Evaluated model performance using R² ≈ 0.85, RMSE ≈ 0.05, and MAE ≈ 0.03, demonstrating strong predictive accuracy against real experimental data.
